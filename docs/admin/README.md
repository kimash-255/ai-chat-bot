# Admin Docs

## Scope

- `/admin` dashboard
- analytics and reporting
- Hugging Face provider configuration
- encrypted dataset dump operations
- sandbox diagnostics
- `cheatcode.md` for admin/recovery verification answers

## Restrictions

- Admin users are redirected to `/admin` after login.
- Admin verification challenge is required during login.

## Analytics Included

- total users
- admin count
- model/provider config count
- coarse geolocation bucket summary
