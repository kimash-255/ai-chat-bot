# Developer Docs

This folder contains developer documentation for architecture, deployment, security, and operations.

## Files

- `architecture.md`: system architecture and module responsibilities.
- `deployment.md`: deployment and environment setup.
- `security.pdf`: security overview in PDF format.
- `frontend/README.md`: frontend structure and UX behavior.
- `backend/README.md`: API and backend architecture.
- `admin/README.md`: admin dashboard and controls.
- `devops/README.md`: ops, env, and logging details.
